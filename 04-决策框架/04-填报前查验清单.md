# 填报前查验清单

> 提交志愿表之前，逐项核对。每项都确认了再点"提交"。

---

## 一、基本信息核对

- [ ] 姓名、身份证号、考生号与准考证一致
- [ ] 科类确认为"历史类"
- [ ] 选考科目确认为"政治、地理"
- [ ] 批次选择正确
- [ ] 志愿数量在规定范围内

---

## 二、学校层面核查

- [ ] **是否在河南历史类招生？** — 有些学校不招历史类考生
- [ ] **招生章程已读？** — 重点看：录取规则、加分政策、身体条件限制、单科成绩要求
- [ ] **专业组的选科要求匹配？** — 必须满足该专业组的选科要求
- [ ] **有没有单科成绩要求？** — 比如英语专业要求英语 ≥ 120
- [ ] **有没有身体条件限制？** — 比如色盲不能报某些专业
- [ ] **校区在哪里？** — 别报到偏远校区/异地校区后才发现
- [ ] **学费多少？** — 中外合作项目可能 3-10 万/年
- [ ] **学制是几年？** — 医学类 5 年，中外合作可能 4+0 或 3+1
- [ ] **该校有没有"专业级差"？** — 大部分学校已取消，但要核实

---

## 三、专业层面核查

- [ ] **该专业的录取位次（不是学校投档位次）已查到？**
- [ ] **专业组内所有专业你都能接受？** — 服从调剂意味着所有专业都可能被录
- [ ] **专业组内有没有你完全不能接受的专业？**
- [ ] **该专业是否有特殊性质？** — 师范类、中外合作、民族班

---

## 四、冲稳保比例核查

- [ ] 大冲院校不超过 5 个
- [ ] 稳的院校至少有 15 个
- [ ] 垫底院校全部位次比你低 15% 以上，绝对安全
- [ ] 垫底院校你能接受去读（不是填了凑数）

---

## 五、排序核查

- [ ] **最喜欢的放在最前面**（不是最稳的放最前面）
- [ ] 冲的院校中，按喜欢程度排序（不是按录取概率排序）
- [ ] 稳的院校中，最喜欢的靠前
- [ ] 保底院校中，最可能去的靠前

---

## 六、特殊类型核查（如适用）

- [ ] **提前批** — 只填比本科批更好的选择
- [ ] **国家专项** — 是否具备资格？
- [ ] **高校专项** — 同上
- [ ] **公费师范生** — 是否了解服务协议？（通常要回生源地从教6年）
- [ ] **优师专项** — 同上

---

## 七、退档风险核查

以下情况会导致退档：

- [ ] **不服从调剂 + 分数不够专业线** — 最常见退档原因
- [ ] **身体条件不符** — 比如色盲报了化学类
- [ ] **单科成绩不达标** — 比如英语专业要求英语 ≥ 120
- [ ] **口语测试未参加** — 部分语言类专业要求外语口试成绩
- [ ] **政审不通过** — 军校、公安类需要政审

> 🚨 平行志愿一旦投档后被退档，本批次后面的志愿全部作废，只能等征集志愿或下一批次。**退档 = 灾难。**

---

## 八、最终确认

- [ ] 志愿表截图/拍照保存
- [ ] 确认码/密码已妥善保管
- [ ] 提交后重新登录检查，确认提交成功
- [ ] 截止时间前 1-2 天提交（不要卡最后几分钟）
- [ ] 提交后每天检查是否有征集志愿通知

---

## 九、提交后

- [ ] 保持手机畅通（录取通知可能需要确认）
- [ ] 关注河南省教育考试院公众号/官网
- [ ] 准备录取后事项（档案、户口迁移、助学贷款等）
