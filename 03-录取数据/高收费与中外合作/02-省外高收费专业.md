# 省外高收费/中外合作专业

> 523 分历史类政史地考生 · 省外公办院校

---

## 1. 核心结论

**523 分在省外的中外合作选择空间远小于省内。** 原因：

1. B区 211 中外合作仍要 580+（宁夏大学合作项目组 582），差 60 分
2. 省属重点师范/财经/语言类的中外合作多在 550+，WebSearch 几乎搜不到 520-540 区间的具体数据
3. 能搜到的省外公办中外合作（海南热带海洋学院 472、桂林旅游学院 473、咸阳师范学院 485）学校层次偏低
4. 和省内一样的规律：工科院校中外合作只招物理类

**务实结论：主力冲省内，省外只做保底补充。**

### 省外怎么找

省外院校的官网数据搜索引擎很难覆盖，建议手工查：

1. 打开目标学校招生网 → "历年分数"栏目
2. 选"河南省" → "历史类/文科" → 看是否有"中外合作"专业组
3. 记录：专业名、最低分、最低位次、学费、选科要求
4. 如果有 520-540 区间的，补入本文件主表

**重点盯三类学校**：
- 东北省属师范（气候冷门 → 分数可能偏低）
- 西北省属院校（地域冷门 → 分数可能偏低）
- B 区省属综合类（同上）
