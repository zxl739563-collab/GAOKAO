## 判断标准

| 与 523 分差距 | 判断 |
|--------------|------|
| 低于 523 分 10 分以上 | 稳/保 |
| 低于 523 分 0-10 分 | 冲稳之间 |
| 高于 523 分 0-10 分 | 可冲 |
| 高于 523 分 10-20 分 | 偏冲/极限冲 |
| 高于 523 分 20 分以上 | 不建议重点押 |
| 高于 523 分 40 分以上 | 基本不现实 |

> 最终需结合位次、招生人数、专业冷热、学费综合判断。
