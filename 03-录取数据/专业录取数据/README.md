# 专业录取数据 — 模板与来源

## 为什么需要专业录取数据

学校的投档线是全校最低分，但同一学校不同专业的分差可以高达 20-40 分。

举例：
- 武汉大学 2024 年在河南文科投档线 637 分
- 但法学专业录取线可能是 645 分（高出 8 分）
- 而哲学专业可能只要 637 分（刚好压投档线）

**只参考学校投档线，很容易被调剂到不喜欢的专业。**

```
|------|------|------|--------|----------|--------|----------|----------|------|
| 2025 |      |      |        |          |        |          |          |      |
```

---

## 数据来源

| 来源 | 获取方式 | 可靠性 |
|------|----------|--------|
| 各校招生网"历年分数"栏目 | 搜索"XX大学XX专业河南录取分数" | ⭐⭐⭐⭐⭐ |
| 河南省教育考试院 | heao.gov.cn | ⭐⭐⭐⭐⭐ |
| 高考志愿填报APP | 优志愿等（付费功能） | ⭐⭐⭐ |

---

## 建议采集方式

1. 确定目标专业方向（3-5个）
2. 确定目标院校（15-30所）
3. 逐一访问目标院校招生网，搜索该专业的河南录取分数
4. 按模板填入本目录
5. 按院校命名文件（如 `武汉大学.md`、`郑州大学.md`）

---

## 关键提醒

- **平均分比最低分更有参考价值**（最低分可能是特殊类型招生拉低的）
- **招生人数少的专业波动大**（招2人的专业，去年位次参考价值有限）
- 关注**专业是否在历史类招生**，新高考后部分文理兼招专业可能转向物理类
