## 一、中国高校层次金字塔（文科视角）

```
        ╱ 北大 人大 复旦 南大 ╲          ← 文科顶尖，全省前100
       ╱  武大 南开 中大 厦大  ╲         ← 文科强校，全省前500
      ╱ 北师大 华师大 川大 山大 ╲        ← 文科强劲，全省前1000
     ╱  中政法 央财 上财 中传  ╲        ← 行业顶尖211，全省前1500
    ╱   郑大 西南 华中师大 南师大 ╲      ← 中坚211，全省前8000
   ╱    河大 西北 湖师大 东北师大  ╲     ← 性价比211，全省前15000
  ╱     省内省属重点 + 省外双非  ╲      ← 本科中坚，全省前30000
 ╱       地市级学院 + 民办本科   ╲      ← 本科保底
```

> ⚠️ 位次是粗略估算，新高考后以实际数据为准。金字塔只示意层次关系。

## 四、文科生选学校的基本逻辑

1. **985标签是硬通货** — 文科就业高度依赖学校牌子，考选调、考研、进国企都会用到
2. **行业211 > 综合985弱势文科** — 中政法的法学 > 某些985的法学
3. **省内好学校性价比极高** — 河南考生报郑大、河大：招生人数多（几百人），位次稳定，离家近
4. **不要只盯着北上广** — 成都、武汉、西安、长沙、南京都有好学校，生活成本低且分没那么高
5. **考虑读研路径** — 本科不是终点，读研可以换学校、换城市、换专业
