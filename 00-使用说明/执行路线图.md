## 🔴 出分前待做（现在）

### 第一优先：搞清楚自己想学什么
- [ ] 通读 [[00-文科专业全景图]]
- [ ] 逐个读 10 个专业详情文件，标记 **3-5 个感兴趣的方向**
- [ ] 在 [[01-个人偏好梳理]] 中填入初步想法

### 第二优先：了解学校格局
- [ ] 通读 [[00-学校分层总览]]
- [ ] 对金字塔各层有概念（哪些学校在哪个层次）

### 第三优先：确认选科限制（新高考关键）
- [ ] 在感兴趣的专业上标注 **选科要求**（首选科目 + 再选科目）
- [ ] 确认你的组合（历史+政史地）能报哪些、不能报哪些
- [ ] 特别注意：护理学、中医学、心理学等传统「文科友好专业」，新高考下可能因再选科目要求而受限

### 第四优先：确认特殊类型资格
- [ ] 读 [[特殊类型招生]]
- [ ] 确认户籍是否在国家专项实施区域
- [ ] 确认是否为农村户籍（涉及高校专项）
- [ ] 考虑是否报考公费师范生

### 数据采集（AI 辅助）
- [ ] 获取**选科要求数据**：河南文科可报专业的完整选科限制表
- [ ] 获取**批次线历史数据**：河南省 2019-2025 各批次线
- [ ] 获取**投档数据**：985/211/省内院校在河南历史类的历年投档分/位次
- [ ] 运行招生章程爬虫获取目标学校的完整章程

### 有空再做：补学校详情
- [ ] 用 [[院校模板]] 的格式，为你感兴趣的 10-20 所学校建详情页
- [ ] 放到 `02-学校库/` 目录下
